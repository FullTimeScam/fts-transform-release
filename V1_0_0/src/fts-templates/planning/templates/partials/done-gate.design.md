### 디자인 gate (필수)

다음 체크리스트를 전부 충족해야 한다. **하나라도 미충족이면 done 중단.**

- [ ] 모든 화면·컴포넌트에 외부 링크(`{{DESIGN_TOOL}}`) 연결
- [ ] 에셋 저장 경로(`{{ASSET_PATH}}`) 가 유효하고 파일 누락 없음
- [ ] 원본 파일과 내보내기 파일이 분리 보관
- [ ] WCAG {{WCAG_LEVEL}} 기준 접근성 체크 완료 (대비비·포커스·대체 텍스트·터치 타겟)
- [ ] 디자인 시스템 토큰(색·타이포·간격) 과 불일치 없음
- [ ] 다크 모드·반응형 등 요구된 변형이 함께 제공
- [ ] 리뷰어 1 명 이상 승인
